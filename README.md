# iONE Speed Monitor

**Version:** 1.0  
**Author:** iONE  

---

## Overview

**iONE Speed Monitor** is a lightweight Windows utility that displays **real-time network download and upload speeds**.  
It runs as a small, always-on-top window, designed for **Windows PCs and laptops**.  
Unlike traditional system tray monitors, it shows the speeds directly in a floating window for instant visibility.  

**Key Features:**
- Shows real-time download and upload speeds
- Floating, always-on-top display
- Minimal CPU and memory usage
- Optional “Run on Windows Startup”
- Clean, simple interface — no clutter

---

## Supported Devices

- Windows 10 / 11  
- Desktop PCs and laptops with a network interface  

> Note: This tool is **not designed for mobile devices** or non-Windows platforms.

---

## Installation

1. Download the installer from GitHub:  
   [Download iONE Speed Monitor 1.2](https://github.com/irazirfan/Speed_Monitor/releases/download/v1.2/iONE_SM_Setup.exe)  

2. Run the installer (`iONE_SM_Setup.exe`) and follow the on-screen instructions.  

3. Optional: On the final installation screen, check **“Launch iONE Speed Monitor”** to start it immediately.  

4. The application will appear as a small floating window displaying your network speed.

---

## Usage

- The speed window will always remain on top.  
- Right-click the window to access options such as:
  - Exit the application  
  - About / Credits  

- Speeds are displayed in **B/s, KB/s, or MB/s** depending on your network usage.
